Capture leads profissionalmente com modal customizável e integração direta com RankiTO CRM.

== Description ==

O Rankito LeadGen é um plugin completo para captura de leads que se integra diretamente com seu CRM RankiTO. Com um modal altamente customizável, você pode capturar informações de potenciais clientes de forma profissional e elegante.

**Principais Recursos:**

* ✅ Modal totalmente customizável (cores, tamanhos, layouts)
* ✅ Campos dinâmicos e personalizáveis
* ✅ Integração direta com RankiTO CRM
* ✅ Validação client-side em tempo real
* ✅ Captura automática de UTMs e dados de navegação
* ✅ Botão flutuante configurável
* ✅ Shortcodes para inserção manual
* ✅ Responsivo e acessível (WCAG AA)
* ✅ Prevenção de leads duplicados
* ✅ Mensagens de sucesso/erro personalizáveis

**Como Funciona:**

1. Instale e ative o plugin
2. Configure a URL da API e o Token de autenticação (fornecido pelo RankiTO)
3. Personalize o visual e os campos do formulário
4. Publique o botão flutuante ou use shortcodes
5. Os leads são enviados automaticamente para seu CRM

**Shortcodes Disponíveis:**

`[rankito_button text="Fale Conosco"]` - Insere um botão que abre o modal
`[rankito_link text="Clique aqui"]` - Insere um link que abre o modal

== Installation ==

1. Faça upload do plugin via WordPress Admin → Plugins → Adicionar Novo
2. Ou extraia os arquivos na pasta `wp-content/plugins/rankito-leadgen`
3. Ative o plugin pelo menu "Plugins" do WordPress
4. Acesse "Rankito LeadGen" no menu admin para configurar

== Frequently Asked Questions ==

= Preciso ter conta no RankiTO? =

Sim, você precisa de uma conta ativa no RankiTO para obter a URL da API e o Token de autenticação.

= Os leads são salvos no WordPress? =

Não, os leads são enviados diretamente para o CRM RankiTO via API segura.

= Posso personalizar as cores e o design? =

Sim! O plugin oferece controle total sobre cores, tamanhos, layouts e mensagens.

= É compatível com outros plugins de formulário? =

Sim, o Rankito LeadGen funciona independentemente e não interfere com outros plugins.

= Como testar a integração? =

No painel admin do plugin, há um botão "Testar Conexão" que valida sua configuração.
